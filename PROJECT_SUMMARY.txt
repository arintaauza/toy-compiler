WHAT IS LUNA?
================================================================================

Luna is a statically-typed, C-like programming language with:

- Static typing: int, float, bool, string, void
- Variables: let x: int = 42;
- Functions: fn add(a: int, b: int) -> int { return a + b; }
- Control flow: if/else, while loops
- Operators: +, -, *, /, %, ==, !=, <, >, <=, >=, &&, ||, !
- Built-in: print() function
- Comments: // single-line, /* multi-line */

Example:
```luna
fn fibonacci(n: int) -> int {
    if n <= 1 { return n; }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

fn main() -> int {
    print(fibonacci(10));  // Prints: 55
    return 0;
}
```

COMPILER ARCHITECTURE
================================================================================

Source Code (.luna)
       |
       v
[Phase 1] Lexer -----------------> Tokens
       |
       v
[Phase 2] Parser ----------------> Abstract Syntax Tree (AST)
       |
       v
[Phase 3] Semantic Analyzer -----> Type-checked AST + Symbol Table
       |
       v
[Phase 4] IR Generator ----------> SSA-based IR + Control Flow Graph
       |
       v
[Phase 5] Optimizer -------------> Optimized IR
       |
       +---> [Phase 6] x86-64 Codegen ---> Native Binary
       |
       +---> [Phase 7] LLVM Codegen -----> LLVM IR ---> Native Binary

[Phase 8] CLI Tool (luna.py) - Professional command-line interface

================================================================================
USAGE
================================================================================

# Compile and run a Luna program
python luna.py examples/fibonacci.luna

# With optimizations
python luna.py -O examples/fibonacci.luna

# Show compilation stages
python luna.py --tokens examples/hello_world.luna    # Show tokens
python luna.py --ast examples/hello_world.luna       # Show AST
python luna.py --ir examples/hello_world.luna        # Show IR
python luna.py --ir-opt examples/hello_world.luna    # Show optimized IR
python luna.py --asm examples/hello_world.luna       # Show x86-64 assembly
python luna.py --llvm examples/hello_world.luna      # Show LLVM IR

# Generate output file
python luna.py -o output.s --asm examples/fibonacci.luna

# Verbose mode (shows timing stats)
python luna.py -v examples/fibonacci.luna

# Use LLVM backend
python luna.py --backend llvm examples/fibonacci.luna

IMPLEMENTATION HIGHLIGHTS
================================================================================

Type System:
- Static typing with int, float, bool, string, void
- Type checking for all operations
- No implicit type conversions

SSA-based IR:
- Each variable assigned exactly once
- Versioned variables (x_0, x_1, x_2)
- Phi functions at control flow joins

Optimizations:
- Constant folding (2+3 -> 5)
- Dead code elimination
- Common subexpression elimination
- Copy propagation
- Algebraic simplification (x*1 -> x)
- Control flow optimization

Code Generation:
- x86-64 AT&T syntax (macOS compatible)
- System V AMD64 ABI
- LLVM IR via llvmlite with JIT execution
